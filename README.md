# Voice Koder

> Control IntellJ IDEA with voice commands

Uses **OpenAI Whisper** to convert speech to text.
And then feed text into **pyautogui** to control the **IntelliJ IDE**.

![](docs/app-icon.png)

## Setup

```
 make setup
```
